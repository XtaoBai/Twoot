# twoot

利用 Vue 框架构建的微博分享评论点赞网站，当前仅涵盖前端部分与文字收送，后端与注册认证系统待更新。本分支采用的是 Vue-3.0 版本。


项目打包构建过程：

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
